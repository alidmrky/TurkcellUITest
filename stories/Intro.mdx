# UI Bileşen Kütüphaneleri Karşılaştırması

Bu Storybook projesi, iki farklı UI bileşen yaklaşımını karşılaştırmak için oluşturulmuştur:

## 🎨 shadcn/ui Yaklaşımı

**shadcn/ui**, modern React uygulamaları için hazır bileşenler sunan bir kütüphanedir.

### Özellikler:

- ✅ **Copy & Paste** yaklaşımı - bileşenleri projenize kopyalayabilirsiniz
- ✅ **Tailwind CSS** ile tam entegrasyon
- ✅ **Radix UI** primitive'leri üzerine kurulu
- ✅ **TypeScript** desteği
- ✅ **Özelleştirilebilir** - istediğiniz gibi değiştirebilirsiniz
- ✅ **Zero dependencies** - sadece ihtiyacınız olan bileşenleri ekleyin

### Avantajları:

- Hızlı geliştirme
- Tam kontrol
- Küçük bundle boyutu
- Modern tasarım

### Kullanım Alanları:

- Yeni projeler
- Hızlı prototipleme
- Özelleştirme gerektiren projeler

## ⚡ Radix UI Yaklaşımı

**Radix UI**, erişilebilir ve özelleştirilebilir UI primitive'leri sağlar.

### Özellikler:

- ✅ **Headless** - stil olmadan sadece davranış
- ✅ **Erişilebilirlik** odaklı
- ✅ **CSS Modules** desteği
- ✅ **TypeScript** desteği
- ✅ **Tam kontrol** - istediğiniz gibi stillendirebilirsiniz
- ✅ **Primitive'ler** - temel bileşenler

### Avantajları:

- Maksimum esneklik
- Erişilebilirlik garantisi
- Küçük bundle boyutu
- Öğrenme değeri yüksek

### Kullanım Alanları:

- Karmaşık özelleştirmeler
- Erişilebilirlik öncelikli projeler
- Öğrenme amaçlı projeler

## 🔍 Karşılaştırma

| Özellik             | shadcn/ui | Radix UI  |
| ------------------- | --------- | --------- |
| **Kurulum**         | Kolay     | Orta      |
| **Özelleştirme**    | Yüksek    | Maksimum  |
| **Erişilebilirlik** | İyi       | Mükemmel  |
| **Bundle Boyutu**   | Küçük     | Çok Küçük |
| **Öğrenme Eğrisi**  | Düşük     | Orta      |
| **Hız**             | Hızlı     | Orta      |

## 🚀 Hangi Yaklaşımı Seçmeli?

### shadcn/ui Seçin Eğer:
